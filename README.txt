Developed an intelligent ground mobile robot that leverages advanced camera vision and digital sensors to autonomously detect and approach user-specified colored balloons for targeted popping.
The system incorporates a servo motor-operated arm for accurate balloon interaction, and features custom C programming using the KIPR Wombat library for seamless navigation, object detection, and user input processing, offering an engaging and interactive user experience.
